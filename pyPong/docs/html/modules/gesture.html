pyPong.gesture¶

gesture.py a component of pyPong.py

Derive a signal from a processed webcam image

Copyright (c) 2010 John Graves MIT License: see LICENSE.txt

class gesture.Gesture¶

Return the highest significant selected point on each side of the image.

A webcam image processed with the green screen approach (background subtraction) may have white pixels which can be used to extract a signal from a gesture (hand movement).

This implementation looks along rows of pixels, counting the non-black (white) pixels. Any count over 5 (about finger width) is considered a valid selection.

Due to the imperfect filtering of the green screen approach (artifacts), the scan looks down a few additional rows to see if they are also selected (since a finger tip is usually connected to a hand below ...).

countWhitePixels(image)¶
Steps down from top row of image within 40 pixels from left and right sides. Checks every other row for over 5 white pixels. Sets signal when it finds three selected rows together.
isWhite(x)¶
Actually just tests x > 0. Used in sum(map( function , row of pixels )) with True counting as 1 to count non-black pixels.
