Welcome to pyPong’s documentation!¶

pyPong is a demonstration of webcam gesture-based control of a familiar game (similar to Pong (R) ) and written in Python.

pyPong is part of the output of the Open Allure project.

A collection of short videos about the Open Allure project are available and there is a Google group you can join for updates and discussion.
